# To-Do List React

A simple to-do list app built with React and Vite.

## Live Demo

Once deployed, the app will be available at:
`https://rokaideveloper.github.io/to-do-list-react/`

## GitHub Pages Setup

This repository uses GitHub Actions to automatically build and deploy the app to GitHub Pages on every push to `main`.

**One-time configuration required:**

1. Go to **Settings → Pages** in this repository.
2. Under **Source**, select **GitHub Actions**.
3. Save — the next push to `main` will trigger the deployment automatically.

## Development

```bash
npm ci          # Install dependencies
npm run dev     # Start development server
npm run build   # Build for production (output: dist/)
```
